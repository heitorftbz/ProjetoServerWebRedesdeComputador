# ProjetoServerWebRedesdeComputador

Servidor HTTP – AV2
Um pequeno servidor Web desenvolvido em **Python**, capaz de processar **requisições HTTP** básicas.  
Projeto criado para a **Atividade AV2**, demonstrando os conceitos de **Sockets** e **comunicação cliente-servidor**.

Funcionalidades
- Cria um socket TCP e escuta conexões HTTP.  
- Processa requisições GET de navegadores.  
- Retorna arquivos HTML locais.  
- Responde com “404 Not Found” quando o arquivo não existe.  
- Exemplo funcional com `HelloWorld.html`.

Tecnologias Utilizadas

- Python 3
- Socket Library (padrão do Python)
- HTML5 / CSS3 para a página de teste
